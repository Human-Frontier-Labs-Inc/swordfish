# 007 - Onboarding "What's Next" Items Not Clickable

**Severity:** P2
**Status:** Open
**Date Found:** 2026-04-10
**Affected Pages:** Onboarding step 6

## Symptoms

The "What's Next?" section on the onboarding completion screen lists:
- View your security dashboard
- Review detected threats
- Configure advanced policies
- Add team members

These are plain text, not links. Users would naturally want to click them.

## Recommendation

Make each item a link to the corresponding page (/dashboard, /dashboard/threats, /dashboard/policies, /dashboard/settings).
