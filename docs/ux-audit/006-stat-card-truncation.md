# 006 - Stat Card Labels Truncated on Desktop

**Severity:** P2
**Status:** Open
**Date Found:** 2026-04-10
**Affected Pages:** Dashboard overview

## Symptoms

On desktop (1280px), dashboard stat card labels are truncated with ellipsis:
- "Threats Blocked..." (should be "Threats Blocked (7d)")
- "Quarantine Pen..." (should be "Quarantine Pending")

## Recommendation

Widen the stat cards slightly, reduce font size, or allow text wrapping.
